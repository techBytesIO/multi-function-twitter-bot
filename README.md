# Twitter BOT README

Written in nodeJS

### Bots inculded in this program

* Retweet Bot
	+ Runs every 4 hours
* Favorite Bot
	+ Runs every 6 hours
* Reply Bot
	+ Streamer runs constant
* DM Bot
	+ Streamer runs constant

### Get it running
Add your Twitter API creds to your .env file
